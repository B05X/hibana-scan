## Installation in 3 steps

```bash
# 1. Clone the repo
git clone https://github.com/B05X/hibana-scan.git

# 2. Change into the directory
cd hibana-scan

# 3. Install dependencies
pip install -r requirements.txt

# 4. run
python hibana_scan.py
```
